# Retrofit2_demo



AppFoods - Ứng Dụng Xem Danh Mục Món Ăn (Retrofit & RecyclerView)
Đây là một ứng dụng Android mẫu (Demo) được phát triển để minh họa cách thực hiện các thao tác kết nối mạng (Network) và tải dữ liệu JSON bất đồng bộ (Asynchronous) bằng thư viện Retrofit,
sau đó hiển thị dữ liệu đó bằng RecyclerView và thư viện tải ảnh Glide.



🛠️ Công Nghệ Sử Dụng
Ngôn ngữ: Java

Target SDK: 36 (Android 14)

Min SDK: 23 (Android 6.0 Marshmallow)



Thư viện = Mục đích
Retrofit 2,HTTP Client dùng để thực hiện các yêu cầu GET/POST tới API.
Gson Converter,Chuyển đổi phản hồi JSON từ API thành các đối tượng Java (Model POJO).
OkHttp Logging Interceptor,"Theo dõi và in ra chi tiết Request/Response (JSON Body, Headers) vào Logcat để Debug."
Glide 4.x,Thư viện hiệu quả để tải ảnh từ URL internet và hiển thị chúng trong ImageView.
RecyclerView & CardView,Hiển thị danh sách dữ liệu cuộn theo chiều ngang (LinearLayoutManager.HORIZONTAL).
